# Simon Says Game

A fun and interactive **Simon Says** memory game built using **HTML, CSS, and JavaScript**. The game tests your memory by generating an increasingly longer sequence of colors that you must repeat correctly.

---

## Overview

This project is a browser-based implementation of the classic Simon Says memory game. With each level, a new color is added to the sequence, making the game progressively more challenging.

---

## Features

- Random color sequence generation
- Memory challenge with increasing difficulty
- Level tracking
- Interactive button animations
- Game Over screen with restart functionality
- Responsive and clean user interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## How to Play

1. Press any key to start the game.
2. Watch the color that flashes.
3. Click the same color.
4. Repeat the complete sequence as new colors are added each level.
5. Clicking the wrong color ends the game.
6. Press any key to restart and try to beat your highest level.
